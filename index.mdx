Helpdesk is a production-ready support platform that lets companies manage customer tickets, publish announcements, and maintain a self-serve help center — all from a single multi-tenant system. Whether you're an agent resolving tickets, a company admin managing your team, or a developer integrating via API, Helpdesk has you covered.

## Get started in four steps

<Steps>
  <Step title="Start the environment">
    Clone the repository and bring up all services with Docker Compose.

    ```bash
    git clone https://github.com/Lukehowland/Helpdesk.git
    cd Helpdesk
    docker compose up -d
    docker compose exec app php artisan migrate --seed
    ```
  </Step>
  <Step title="Register and authenticate">
    Create a user account and obtain a JWT access token for API calls.

    ```bash
    curl -X POST http://localhost:8000/api/auth/register \
      -H "Content-Type: application/json" \
      -d '{
        "email": "you@example.com",
        "password": "SecurePass123!",
        "passwordConfirmation": "SecurePass123!",
        "firstName": "Jane",
        "lastName": "Doe",
        "acceptsTerms": true,
        "acceptsPrivacyPolicy": true
      }'
    ```
  </Step>
  <Step title="Create your first ticket">
    Submit a support ticket using your JWT token.

    ```bash
    curl -X POST http://localhost:8000/api/tickets \
      -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "title": "Cannot access my account",
        "description": "I get a 403 error when trying to log in.",
        "category_id": "CATEGORY_UUID"
      }'
    ```
  </Step>
  <Step title="Explore the platform">
    Check the API reference for all available endpoints, or dive into the feature guides to understand announcements, the help center, and company management.
  </Step>
</Steps>
